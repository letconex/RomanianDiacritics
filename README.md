# RomanianDiacritics
OpenOffice/LibreOffice Extension for handling Romanian Diacritics in Writer  
Inserts the Romanian characters with diacritics by using the mouse, replaces ș/Ș and ț/Ț characters with comma bellow by their equivalents with cedilla below or vice versa, replaces all Romanian characters with diacritics by standard ASCII characters.  

Extension for LibreOffice/OpenOffice Writer, GPLv3 license.  
Versions 1.0.0-1.0.2 published on https://extensions.openoffice.org/en/project/romanian-diacritics  
Versions 1.1.0-1.1.1 published on https://www.secarica.ro/en/rou/librarydiacritice  
Versions 1.1.2-1.1.x published on https://extensions.libreoffice.org/en/extensions/show/99553
